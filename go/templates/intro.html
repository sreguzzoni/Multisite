Technologies Used

This section of the website is built using the Go programming language, known for its efficiency and performance. The following technologies are used:

  • Go: A statically typed, compiled programming language designed for simplicity and performance.
  • Tailwind CSS: A utility-first CSS framework for rapid UI development.
  • HTML Templates: Go's built-in HTML template package for server-side rendering of HTML pages.
  • Docker: Containerization technology to package and run applications in isolated environments.
  • Nginx: A web server and reverse proxy server for serving static content and proxying requests to the Go application.
  • AOS (Animate On Scroll): A library to animate elements on your page as you scroll.

Each page is carefully designed to showcase different aspects of the projects and repositories, utilizing modern web development practices to ensure a smooth and responsive user experience.
